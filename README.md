## 🚀 Project Overview
This repository contains the deliverables for my role as a **Student Digital Ambassador**. The project is structured around five core tasks designed to navigate the modern digital world safely and professionally.

### 📂 Task Modules

#### 1. Digital Literacy Awareness Infographic
* Created a visual guide titled **"Digital Literacy for Beginners"** using **Canva**.
* Topics include defining digital literacy, safe internet practices, and maintaining a professional online presence.

#### 2. Student Digital Portfolio
* Established professional profiles on **GitHub**, **LinkedIn**, and **Kaggle**.
* Goals include learning programming fundamentals and building a technical portfolio over four years.

#### 3. Coding and Collaboration Platforms
* Developed a **Digital Literacy Awareness Quiz** using **Google Forms** for peer assessment.
* Completed beginner challenges on **HackerRank** to familiarize with competitive coding.

#### 4. Email Etiquette Guide
* Analyzed a case study on communication failures in group projects.
* Drafted formal emails for academic and professional scenarios, such as requesting extensions and internship inquiries.

#### 5. Cybercrime Awareness
* Researched a **Phishing Case Study** involving a fake "Urgent Semester Fee" campaign.
* Documented prevention strategies, including verifying URLs and enabling **two-factor authentication**.
